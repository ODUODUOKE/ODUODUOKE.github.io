git 拉取远端所有分支

CSDN博客原文参考地址

关键命令代码

git branch -r | grep -v '\->' | while read remote; do git branch --track "${remote#origin/}" "$remote"; done git fetch --all git pull --all

操作步骤截图

1.通过 git bash 窗口输入git clone 命令

2.克隆项目到本地

3.进入项目 jmt_lp 文件夹目录下

4.执行以下命令

git branch -r | grep -v '\->' | while read remote; do git branch --track "${remote#origin/}" "$remote"; done git fetch --all git pull --all

下载所有分支代码
