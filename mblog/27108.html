关系型数据库和非关系型数据库区别

  1. 数据一致性不同
  2. 数据储存方式不同
  3. 扩展方式不同
  4. 对事务的支持不同

常见关系型数据库

  • Oracle
  • Sql Server
  • Mysql
  • Db2

常见非关系型数据库

  • Redis
  • MongoDB
  • HBase
  • Neo4j
