maven - mirrorOf 的坑、多镜像切换（避免一切无厘头报错） 问题描述 Idea配置Repository不生效,导致项目获取不到相应依赖。 原因 maven pom中的repository节点配置没有起作用 写*会覆盖掉所有的，不管是哪个repository，最后都被这个镜像所mirror掉了，导致pom文件中的repository不生效了。 mirrorOf官方说明 * = everythingexternal:* = everything not on the localhost and not file based.repo,repo1 = repo or repo1*,!repo1 = everything except repo1 todo 还是得下来把这个搞懂才行~
